## What is AAYU?

AAYU is an experimental next-generation programming language and intent-driven computing platform designed to make software development more natural, accessible, and productive.

Unlike traditional languages such as Python, Java, C++, Rust, and JavaScript, AAYU focuses on understanding **human intent** rather than forcing developers to think in machine-oriented syntax.
